# ConhecendoEAplicandoSQL
Pequeno projeto com a intenção de conhecer um pouco a linguagem SQL

**Desenvolvido Por** *Arthur Felipe MS*

## Linguagem utilizada.
* *SQL.*

### Desenvolvido  na IDE
* *MySQL Workbench.*

### Conceitos Aplicados. 
* **Criação do Banco de Dados.**

* **Criação de Tabelas.**
* * *Tipos de dados*
* * *Conceito de Chave Primária.*
* * *Conceito de Chave Estrangeira.*
* * *Conceito de Chave Única.*
* * *Relações entre Tabelas.*
* * * 1 p/ M.
* * * 1 p/ M / M p/ 1.
* * * M para M.
* * *Not Null.*
* * *Unsigned.*
* * *AUTO_INCREMENT.* 

* **Inserir Dados nas tabelas.**
* * *Com Select.*

* **Consultar Tabelas.**
* * *Where.*
* * *Oder By.*
* * *ASC/DESC*
* * *Group By*
* * *Joins*

* **Atualizar Tabelas.**
* * *Update.*

* **Deletar Dados/Tabela**

* **Apagar Dados TRUNCATE**
